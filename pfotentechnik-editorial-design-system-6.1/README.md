# PfotenTechnik Editorial Design System 6.1

Enthält Produktkarten V3, kleinere unverzerrte Bilder, 0–100-Scores ohne Sterne, zweizeilige Titel, dreizeilige Beschreibungen, kompaktere Checklisten, mobile Tabellen, einheitliche CTA-Höhen und Sicherheitsabstand für Floating-Elemente.

## Installation

```bash
node pfotentechnik-editorial-design-system-6.1/install-editorial-design-system-6.1.mjs
npm run build:pfotentechnik
```

Danach den Dev-Server neu starten.

## Rollback

```bash
node pfotentechnik-editorial-design-system-6.1/rollback-editorial-design-system-6.1.mjs
```
