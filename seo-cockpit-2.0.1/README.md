# SEO Cockpit 2.0.1

Behebt den Syntaxfehler des Installers 2.0.0.

## Installation

Im Repository-Root:

```bash
rm -rf seo-cockpit-2.0.0
unzip seo-cockpit-2.0.1.zip
node seo-cockpit-2.0.1/install-seo-cockpit-2.0.1.mjs
npm run seo:cockpit
```

Danach:

```bash
cd apps/pfotentechnik
npx astro dev
```

Cockpit:

```text
http://localhost:4321/admin/seo/cockpit/
```
