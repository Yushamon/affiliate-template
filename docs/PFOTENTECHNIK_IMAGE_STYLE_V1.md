# PfotenTechnik Image Style v1.0

> Ergänzung zum **PFOTENTECHNIK_VISUAL_STANDARD_V1.md**. Dieses Dokument
> definiert den Stil aller KI-generierten Bilder und Infografiken für
> Premium Guides.

------------------------------------------------------------------------

# Ziel

Alle Bilder sollen wirken, als kämen sie aus einer einzigen
Designbibliothek.

Der Stil ist:

-   modern
-   hochwertig
-   ruhig
-   vertrauenswürdig
-   informativ
-   wiedererkennbar

Nicht dekorativ, sondern wissensorientiert.

------------------------------------------------------------------------

# Grundstil

-   Editorial Design
-   Minimalistisch
-   Viel Weißraum
-   Hochwertige Vektorillustrationen kombiniert mit realistischen
    Objekten
-   Klare Hierarchie
-   Keine Werbung
-   Keine Wasserzeichen
-   Keine Logos
-   Keine Stockfoto-Optik

Leitbild:

> Apple × National Geographic × veterinärmedizinisches Fachmagazin

------------------------------------------------------------------------

# Farbpalette

## Primär

-   #2E7D32

Verwendung:

-   positive Hinweise
-   Lösungen
-   Empfehlungen

## Sekundär

-   #4F46E5

Verwendung:

-   Hintergrundwissen
-   Expertenboxen

## Warnung

-   #F59E0B

Verwendung:

-   Hinweise
-   Wartung
-   Aufmerksamkeit

## Kritisch

-   #DC2626

Verwendung:

-   Schimmel
-   Defekte
-   Warnungen

## Neutral

-   #F8FAFC
-   #E5E7EB

------------------------------------------------------------------------

# Hintergrund

-   Weiß oder sehr helles Grau
-   Keine Muster
-   Keine Verläufe
-   Viel Freiraum

------------------------------------------------------------------------

# Licht

-   Weiches Tageslicht
-   Natürliche Schatten
-   Keine dramatischen Effekte
-   Kein HDR
-   Keine Lens Flares

------------------------------------------------------------------------

# Perspektive

-   Leicht erhöhte Ansicht (ca. 25°)
-   Keine Fisheye-Effekte
-   Keine extremen Blickwinkel

------------------------------------------------------------------------

# Tiere

-   Natürlich
-   Ruhig
-   Nicht vermenschlicht
-   Keine Kleidung
-   Keine übertriebenen Gesichtsausdrücke

------------------------------------------------------------------------

# Produkte

-   Realistisch
-   Sauber
-   Ohne Logos
-   Ohne Markenkennzeichnung
-   Keine Werbedarstellung

------------------------------------------------------------------------

# Typografie

Wenn Text enthalten ist:

-   maximal fünf Kernaussagen
-   große Schrift
-   hoher Kontrast
-   mobil gut lesbar

------------------------------------------------------------------------

# Icons

-   Einheitlicher Outline-Stil
-   Runde Ecken
-   Dezent
-   Modern

------------------------------------------------------------------------

# Diagramme

Bevorzugte Formen:

-   Ablaufdiagramme
-   Vergleichstabellen
-   Entscheidungsbäume
-   Checklisten
-   Kalender
-   Zeitleisten

Nicht verwenden:

-   Tortendiagramme
-   3D-Diagramme
-   dekorative Grafiken ohne Informationsgewinn

------------------------------------------------------------------------

# Qualitätscheck

Vor jeder Veröffentlichung prüfen:

-   [ ] in unter 3 Sekunden verständlich
-   [ ] mobil gut lesbar
-   [ ] einheitlicher Stil
-   [ ] hoher Informationswert
-   [ ] keine Werbung
-   [ ] keine unnötigen Elemente

------------------------------------------------------------------------

# Master Prompt

``` text
Create a premium editorial knowledge graphic.

Style:
Minimalistic.
Apple-level product design.
National Geographic editorial quality.
Veterinary education aesthetic.

White background.
Large whitespace.
Flat vector illustrations mixed with realistic objects.
Natural colors.
Soft green accents.

Premium magazine layout.
Very high readability.

No logo.
No watermark.
No branding.
No stock photo feeling.

German labels if text appears.

Everything should look like one consistent visual language.

Aspect ratio 16:9 unless specified.
High resolution.
```

------------------------------------------------------------------------

# Arbeitsweise

1.  Immer zuerst den Master Prompt verwenden.
2.  Anschließend nur das Thema ergänzen.
3.  Jede Grafik muss Wissen vermitteln.
4.  Alle Premium Guides verwenden dieselbe visuelle Sprache.
