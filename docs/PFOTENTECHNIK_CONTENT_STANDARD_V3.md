# PfotenTechnik Content Standard v3

## Ziel

Jeder Premium Guide soll die beste deutschsprachige Ressource zu seinem
Thema werden.

## Pflichtbestandteile

-   Aussagekräftiges Frontmatter
-   Quick Answer
-   Inhaltsverzeichnis
-   Auf einen Blick
-   Problemerklärung
-   Praxisanleitung
-   Tabellen
-   Checklisten
-   Information-Gain-Abschnitte
-   Natürliche Produktintegration
-   Interne Verlinkung
-   FAQ
-   Quellen
-   Fazit

## SEO

-   Suchintention vollständig beantworten
-   H2/H3 entlang realer Nutzerfragen
-   Long-Tail-Keywords natürlich integrieren
-   Aussagekräftige Alt-Texte
-   Interne Links zu Hub- und Detailseiten

## E-E-A-T

-   Aussagen begründen
-   Herstellerhinweise berücksichtigen
-   Seriöse Quellen
-   Keine unbelegten Behauptungen

## Produktintegration

-   Nur dort empfehlen, wo Technik ein Problem löst.
-   Keine Werbesprache.
-   Produkte im Fließtext erklären.

## Qualitätscheck

-   Beantwortet der Artikel alle Kernfragen?
-   Enthält er einzigartige Informationen?
-   Gibt es mindestens eine Tabelle und eine Checkliste?
-   Ist der Artikel mobil gut lesbar?
