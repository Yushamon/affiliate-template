# PfotenTechnik UI Polish 4.0

UI Polish 4.0 vereinheitlicht die visuellen Grundlagen der zentralen Seitentypen.

## Enthalten

- einheitliche Kartenradien
- einheitliche Schatten
- bessere vertikale Abschnittsabstände
- einheitliche Eyebrows
- kompakte Icon-Badges statt frei schwebender Zeichen
- balancierte Überschriften
- konsistente CTA- und Buttonformen
- robuste Chips mit Umbruch
- tabellarische Ziffern für Bewertungen
- dezente Bildinteraktionen
- vollständige Dark-Mode-Anpassung der neuen Elemente
- reduzierte Bewegung bei `prefers-reduced-motion`
- mobile Typografie und Abstände

## Betroffene Bereiche

- Premium Blocks
- Kurzantworten
- Entscheidungshilfen
- Info- und Warnkarten
- Gesundheitshinweise
- Product Trust Panel
- Alltagstest
- Conversion Journey
- Herstellerkarten
- Produkt- und Vergleichskarten
- Chips und Bewertungen

## Installation

```bash
node pfotentechnik-ui-polish-4.0/install-ui-polish-4.0.mjs
npm run build:pfotentechnik
```

Danach:

```bash
npm run dev:pfotentechnik
```

## Rollback

```bash
node pfotentechnik-ui-polish-4.0/rollback-ui-polish-4.0.mjs
```
