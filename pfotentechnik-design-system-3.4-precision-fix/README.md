# PfotenTechnik Design System 3.4

Präzisionsfix für die verbleibenden Dark-Mode-Kontrastfehler.

## Installation

```bash
node pfotentechnik-design-system-3.4-precision-fix/install-design-system-3.4.mjs
npm run build:pfotentechnik
npm run dev:pfotentechnik
```

## Korrigiert

- HomeUseCases-Karten
- Homepage-Vergleichs- und Editorial-Karten
- Vergleichs-Empfehlungskarten
- Produkt-Hero und Produkttitel
- Gesamturteil
- Stärken und Schwächen
- technische Daten
- Alltagstest-Karten
- Related-Content-Karten
- mobile Sticky-Bar

## Rollback

```bash
node pfotentechnik-design-system-3.4-precision-fix/rollback-design-system-3.4.mjs
```
