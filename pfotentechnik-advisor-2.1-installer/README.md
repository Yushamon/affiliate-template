# PfotenTechnik Advisor 2.1

Installation im Repository-Root:

```bash
node pfotentechnik-advisor-2.1-installer/install-advisor-2.1.mjs
npm run build:pfotentechnik
```

Integration:
- Eigene Seite: `/berater/futterautomat/`
- Startseite: CTA direkt unter `HomePage`
- Vergleichsseiten: CTA vor `ConversionJourney`

Rollback:
```bash
node pfotentechnik-advisor-2.1-installer/rollback-advisor-2.1.mjs
```
