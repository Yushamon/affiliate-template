# PfotenTechnik Product Renderer 4.1 Premium

Diese Version basiert direkt auf den hochgeladenen aktuellen Dateien und
übernimmt deren komplette Datenlogik.

## Dark Mode

Der Renderer verwendet ein vollständiges, abgestimmtes Dark-Mode-System:

- dunkler Seiten- und Kartenhintergrund
- keine festen weißen Komponentenflächen
- abgestufte dunkle Oberflächen
- kontrastreiche Textfarben
- reduzierte Schatten im Dark Mode
- angepasste Rahmen, Buttons, Quick Facts, Buybox und Galerie
- grüne Akzentfarben mit ausreichendem Kontrast

## Weitere Änderungen

- Bewertung ausschließlich als `xx/100`
- keine Sterne
- keine sichtbaren `editorial-review`-Werte
- Primärfakten als Highlight-Karten
- keine abgeschnittenen Inhalte
- Premium Hero, Galerie, Buybox und Karten
- doppelte Trust-Leiste entfernt
- alte DOM-Enhancer und Präsentations-Fixes entfernt
- responsive Mobile-, Tablet- und Desktop-Layouts
- Buildprüfung und automatischer Rollback

## Installation

Im Repository-Root:

```bash
node <entpackter-ordner>/install.mjs
```
