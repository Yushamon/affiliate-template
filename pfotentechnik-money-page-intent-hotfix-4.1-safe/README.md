# PfotenTechnik Money Page Intent Hotfix 4.1

Der Fix löst drei Punkte:

- Money-Pages ohne exakten Größen-Treffer zeigen die beste verfügbare Option mit klarer Einschränkung.
- Der Vergleichsbereich verwendet die kurze Benennung **Passende Modelle vergleichen** und den Button **Zum Vergleich**.
- Oberhalb des langen Ratgeberteils erscheint ein Zwei-Wege-Einstieg:
  - **Zum Vergleich**
  - **Kriterien prüfen**

So werden kaufbereite und informationsorientierte Besucher gleichzeitig abgeholt.

## Installation

```bash
python3 apply-pfotentechnik-money-page-intent-hotfix-4.1-safe.py
```
