# PfotenTechnik Advisor 2.0

Premium-Kaufberater für Futterautomaten mit:

- mobile-first UX
- sechs geführten Schritten
- Live-Zwischenauswertung
- qualitativen Passungsstufen
- harten Ausschlussregeln
- drei nachvollziehbaren Empfehlungen
- Vergleich nur der Unterschiede
- URL-Sharing
- lokalem Wiederherstellen der Auswahl
- passenden Ratgebern
- barrierearmer Tastaturbedienung
- reduzierten Animationen bei `prefers-reduced-motion`

## Installation

Im Repository-Root:

```bash
node install-advisor-2.mjs
npm run build:pfotentechnik
```

Danach:

```text
/futterautomat-berater/
```

Der Installer überschreibt keine vorhandenen Dateien.

## Empfohlener Rollout

1. Route installieren und Build prüfen.
2. Intern testen.
3. Link im Cornerstone ergänzen.
4. Link auf der Startseite ergänzen.
5. Erst danach Tracking aktivieren.
