# PfotenTechnik Comparison Premium UX 3.0.0

Enthalten:

- Premium Desktop-Layout mit sticky Gewinnerkarte
- kompakte Bereichsnavigation
- Tabellen- und Kartenansicht
- mobile Filter als Bottom Sheet
- mobile Swipe-Karten
- semantische Light-/Dark-Mode-Tokens
- manuelles Theme und `prefers-color-scheme`
- Focus States und Reduced Motion
- Backups und Dry Run

## Installation

Im Repository-Root:

```bash
node ./pfotentechnik-comparison-premium-ux-3.0.0/apply-pfotentechnik-comparison-premium-ux-3.0.0.mjs --check
node ./pfotentechnik-comparison-premium-ux-3.0.0/apply-pfotentechnik-comparison-premium-ux-3.0.0.mjs
npm run build:pfotentechnik
```
