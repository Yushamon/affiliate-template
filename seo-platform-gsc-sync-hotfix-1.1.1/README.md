Im Repository-Root:

```bash
node seo-platform-gsc-sync-hotfix-1.1.1/install-gsc-sync-hotfix-1.1.1.mjs

```

Der Installer legt vor der Änderung eine Sicherungskopie mit der Endung `.bak-1.1.1` an.
