# SEO Dashboard Foundation 1.0

Installation im Repository-Root:

```bash
node /Pfad/zu/seo-dashboard-foundation-1.0/install.mjs
```

Enthalten: Astro-Admin unter `/admin/seo/`, responsive Navigation, Dark Mode, KPI-/Status-/Empfehlungs-/Tabellen-Widgets, globale Suche, GSC-Report-Import, erste regelbasierte Empfehlungen, Report-Übersicht sowie Build-Prüfung mit Rollback.

**Sicherheit:** `noindex,nofollow` ist gesetzt. Die Route ist noch nicht authentifiziert und sollte vor öffentlichem Einsatz über den Hoster geschützt werden.
