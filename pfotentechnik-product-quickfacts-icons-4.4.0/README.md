# PfotenTechnik Quick-Facts Icons 4.4.0

Behebt die falsche Position der Icons und der hellgrünen Icon-Quadrate in den Produkt-Quick-Facts.

- stabile zweispaltige Grid-Struktur
- Iconbox vollständig innerhalb der Karte
- keine negativen Offsets oder absoluten Positionen
- einheitliche Iconbox-Größe
- SVG exakt mittig
- saubere mobile Ausrichtung bei langen Texten

## Windows

```powershell
py .\pfotentechnik-product-quickfacts-icons-4.4.0\apply-pfotentechnik-product-quickfacts-icons-4.4.0.py
```

Falls nötig:

```powershell
npm run build:pfotentechnik
```
